## Quick Start

Fetch the list of supported chains with a single call:

```bash
curl -X GET "https://api.shapeshift.com/v1/chains" \
  -H "Content-Type: application/json"
```

Get swap rates across all available swappers:

```bash
curl -X GET "https://api.shapeshift.com/v1/swap/rates?\
sellAssetId=eip155:1/slip44:60&\
buyAssetId=eip155:1/erc20:0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48&\
sellAmountCryptoBaseUnit=1000000000000000000" \
  -H "X-API-Key: your-api-key"
```

## Key Features

- **Multi-Chain Support** — Access multiple blockchains through a single, unified API
- **CAIP-19 Asset IDs** — Industry-standard asset identification across all chains
- **Real-Time Rates** — Live swap rates with optimized routing across DEXs
- **RESTful API** — Clean, predictable endpoints with consistent JSON responses
- **Interactive Playground** — Test every endpoint directly from the docs
