## Quick Start

### Installation

<CodeGroup>
```bash npm
npm install @shapeshift/api-client
```

```bash yarn
yarn add @shapeshift/api-client
```

```bash pnpm
pnpm add @shapeshift/api-client
```
</CodeGroup>

### Basic Setup

Initialize the client with your API key and start querying chains, assets, and swap rates:

```typescript
import { ShapeShiftAPI } from "@shapeshift/api-client";

const client = new ShapeShiftAPI({
  apiKey: "your-api-key",
});

// List all supported chains
const chains = await client.getChains();
console.log(chains);

// Get a swap quote
const quote = await client.getSwapQuote({
  sellAsset: "eip155:1/slip44:60",       // ETH on Ethereum
  buyAsset: "eip155:137/slip44:966",      // MATIC on Polygon
  sellAmount: "1000000000000000000",       // 1 ETH in wei
});
console.log(quote);
```

## Key Features

- **Multi-Chain Support** — Access 20+ blockchains through a single, unified API
- **CAIP-19 Asset IDs** — Industry-standard asset identification across all chains
- **Real-Time Rates** — Live swap rates with optimized routing across DEXs
- **RESTful API** — Clean, predictable endpoints with consistent JSON responses
- **Interactive Playground** — Test every endpoint directly from the docs
