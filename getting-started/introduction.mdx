## Your First Call

Fetch the list of supported chains:

```bash
curl -X GET "https://api.shapeshift.com/v1/chains" \
  -H "Content-Type: application/json"
```

## Integration Workflow

<Steps>
  <Step title="Get your API key">
    Request access to receive your `X-API-Key` credentials.
  </Step>
  <Step title="Query supported assets">
    Use the [Assets endpoints](/api-reference/overview#assets) to find CAIP-19 asset IDs for your trading pairs.
  </Step>
  <Step title="Fetch swap rates">
    Call `GET /v1/swap/rates` to compare rates across multiple DEX aggregators.
  </Step>
  <Step title="Generate a quote">
    Use `POST /v1/swap/quote` to get executable transaction data with slippage protection.
  </Step>
  <Step title="Execute on-chain">
    Submit the returned transaction object to the blockchain via your wallet or web3 provider.
  </Step>
</Steps>
