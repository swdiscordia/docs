## Basic Usage

```typescript
import { ShapeShiftClient } from '@shapeshift/api-client';

const client = new ShapeShiftClient({
  apiKey: 'your-api-key',
});

// Get swap rates
const rates = await client.swap.getRates({
  sellAssetId: 'eip155:1/slip44:60',
  buyAssetId: 'eip155:1/erc20:0xa0b86991c...',
  sellAmountCryptoBaseUnit: '1000000000000000000',
});

console.log(rates);
```

## Integration Workflow

<Steps>
  <Step title="Get your API key">
    Request access to receive your \`X-API-Key\` credentials from the [developer portal](https://shapeshift.com/developers).
  </Step>
  <Step title="Query supported assets">
    Use the [Assets endpoints](/api-reference/overview#assets) to find CAIP-19 asset IDs for your trading pairs.
  </Step>
  <Step title="Fetch swap rates">
    Call \`GET /v1/swap/rates\` to compare rates across multiple DEX aggregators.
  </Step>
  <Step title="Generate a quote">
    Use \`POST /v1/swap/quote\` to get executable transaction data with gas estimates and slippage protection.
  </Step>
  <Step title="Execute on-chain">
    Submit the returned transaction object to the blockchain via your wallet or web3 provider.
  </Step>
</Steps>
