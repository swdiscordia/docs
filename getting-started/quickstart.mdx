## Your First API Call

Let's start by fetching the list of supported blockchains:

<CodeGroup>
```bash cURL
curl -X GET "https://api.shapeshift.com/v1/chains" \
  -H "Content-Type: application/json"
```

```python Python
import requests

response = requests.get(
    "https://api.shapeshift.com/v1/chains",
    headers={
        "Content-Type": "application/json"
    }
)

data = response.json()
print(data)
```

```javascript JavaScript
const response = await fetch("https://api.shapeshift.com/v1/chains", {
  headers: {
    "Content-Type": "application/json",
  },
});

const data = await response.json();
console.log(data);
```
</CodeGroup>

</Info>

## Execute a Swap

Now let's do something more interesting â€” get a swap quote:

<Steps>
  <Step title="Get rates">
    First, compare rates across aggregators:
    ```bash
    curl -X GET "https://api.shapeshift.com/v1/swap/rates?\
    sellAssetId=eip155:1/slip44:60&\
    buyAssetId=eip155:1/erc20:0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48&\
    sellAmountCryptoBaseUnit=1000000000000000000" \
      -H "X-API-Key: your-api-key"
    ```
  </Step>
  <Step title="Generate a quote">
    Pick the best rate and generate executable transaction data:
    ```bash
    curl -X POST "https://api.shapeshift.com/v1/swap/quote" \
      -H "X-API-Key: your-api-key" \
      -H "Content-Type: application/json" \
      -d '{
        "sellAssetId": "eip155:1/slip44:60",
        "buyAssetId": "bip122:000000000019d6689c085ae165831e93/slip44:0",
        "sellAmountCryptoBaseUnit": "1000000000000000000",
        "swapperName": "Relay",
        "receiveAddress": "bc1q...",
        "sendAddress": "0x..."
      }'
    ```
  </Step>
  <Step title="Submit on-chain">
    Use the returned \`steps[].transactionData\` object to submit via your wallet or web3 provider.
  </Step>
</Steps>
