## API Key

Include your API key in the `X-API-Key` header for all swap endpoints. Chains and assets endpoints are publicly accessible without authentication.

```bash
curl -X GET "https://api.shapeshift.com/v1/swap/rates?sellAssetId=eip155:1/slip44:60&buyAssetId=eip155:1/erc20:0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48&sellAmountCryptoBaseUnit=1000000000000000000" \
  -H "X-API-Key: your-api-key"
```

<Warning>
  **Keep your API key secret.** Never expose it in client-side code, public repositories, or browser requests.
</Warning>
