# ShapeShift API Documentation

Official documentation for the ShapeShift cross-chain swap API, built with [Mintlify](https://mintlify.com).

**Live docs:** [shapeshift.mintlify.app](https://shapeshift.mintlify.app)

## Structure

```
├── docs.json              # Mintlify configuration
├── openapi.yaml           # OpenAPI 3.1.0 specification
├── index.mdx              # Landing page
├── getting-started/       # Guides
│   ├── quickstart.mdx
│   ├── authentication.mdx
│   └── asset-ids.mdx
├── api-reference/         # API reference
│   └── overview.mdx
└── images/                # Logos & assets
    ├── logo-dark.svg
    ├── logo-light.svg
    └── favicon.svg
```

## Local Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mintlify):

```bash
npm i -g mintlify
```

Preview locally:

```bash
mintlify dev
```
