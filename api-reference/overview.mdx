## Authentication

Swap endpoints require key-based authentication. Pass your API key via the `X-API-Key` header. Chains and assets endpoints are publicly accessible.

```bash
curl -X GET "https://api.shapeshift.com/v1/swap/rates?sellAssetId=...&buyAssetId=...&sellAmountCryptoBaseUnit=..." \
  -H "X-API-Key: your-api-key"
```

## Response Format

Successful responses return the data directly. Error responses follow this format:

```json
{
  "error": {
    "code": "ERROR_CODE",
    "message": "Human-readable error description"
  }
}
```

## Endpoints Overview

### Chains

Query supported blockchains and their metadata.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/v1/chains` | List all supported blockchains |
| `GET` | `/v1/chains/count` | Get total chain count |

### Assets

Browse and search supported assets using CAIP-19 identifiers.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/v1/assets` | List supported assets with filtering and pagination |
| `GET` | `/v1/assets/{assetId}` | Get detailed info about a specific asset |

### Swaps

Get real-time swap rates and generate optimized quotes.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/v1/swap/rates` | Get current swap rates between asset pairs |
| `POST` | `/v1/swap/quote` | Generate an optimized swap quote |
